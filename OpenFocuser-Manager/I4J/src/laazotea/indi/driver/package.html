Provides the classes necessary to create INDI Drivers, their Properties and Elements to which the INDI Clients may connect.

Usually a INDI Driver will extend the class INDIDriver and will add as many INDIProperty as needed.

Please check laazotea.indi.driver.examples.INDIElTiempoDriver, for a simple example on the use of the library.

If you want to use it with the standard not I4J indiserver application you will have tu use a shell wrapper. A example one may be a launch.sh file with the following contents:

#!/bin/bash
java -cp ./dist/I4JDriver.jar INDIDriverRunner laazotea.indi.driver.examples.INDIElTiempoDriver

And the command to launch it would be:

> indiserver ./launch.sh @see INDIDriver @see laazotea.indi.driver.examples.INDIElTiempoDriver
